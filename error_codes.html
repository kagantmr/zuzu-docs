System call returns

If an error occurs during system call service, the kernel will return a negative value in r0. You can use this table to deduce which error was returned.

Value Name Meaning
0 ZUZU_OK Success
-1 ERR_NOPERM Operation not permitted
-2 ERR_NOENT Resource or name not found
-3 ERR_BUSY Resource busy, try again
-4 ERR_NOMEM Out of memory
-5 ERR_BADFORM Bad handle, invalid or wrong slot number
-6 ERR_BADARG Invalid argument
-7 ERR_NOMATCH Syscall not implemented
-8 ERR_BADPTR Bad pointer: out of userspace range or would cause a page fault
-9 ERR_DEAD Endpoint destroyed while a thread was waiting on it
-10 ERR_TIMEOUT Message request or operation timed out
